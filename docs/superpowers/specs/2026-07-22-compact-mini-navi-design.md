# コンパクトみになび設計

## 目的

PoE画面の視界を妨げずに攻略案内を読めるよう、ExileUI風のコンパクトなみになび表示を追加する。キャンペーン中の案内を省略せず、日本語を含む本文を見切れなく表示する。

## 表示モード

- 設定から `標準` と `コンパクト` を選択できる。
- 標準は既存のみになびの見た目と動作を維持する。
- コンパクトは初回に画面下中央へ表示し、横幅は約390pxを初期値とする。
- 各モードは位置とサイズを別々に保存する。モード切替は他方の配置を変更しない。
- 両モードとも、ドラッグ移動とリサイズを利用できる。

## コンパクト表示

- 案内は黒背景の横長パネルに箇条書きで表示する。
- 表示行数の上限は設けず、案内文の量に合わせて高さを自動調整する。
- 本文幅は固定の差し引き値ではなく、Qtレイアウトの実際の利用可能幅から、矢印・レベル表示・余白・サイズグリップを差し引いて算出する。
- 日本語本文は折り返して全文を表示する。矢印、レベル表示、本文は重ならない。
- パネルが画面高を超える場合だけ、利用可能な画面領域内に収める。

## 設定と互換性

- 既存の `mini_guide_overlay` 設定は標準モードの設定として扱う。
- コンパクト用の位置・サイズは専用の設定キーへ保存する。
- 既存ユーザーの位置、サイズ、透過率、ロック状態を変更しない。

## 検証

- モード切替とモード別の位置・サイズ保存をテストする。
- 長い日本語案内で高さが内容量に追従し、本文が利用可能幅を超えないことをテストする。
- ロック中、ロック解除時のサイズグリップ表示、最小幅、画面高制限をテストする。
- 既存テストを含めて `pytest` を実行する。
