# ジェムショップ購入案内（みになび）設計

## 目的

PoBから現在Actでショップ購入するジェムがある場合、町に入ったプレイヤーへみになびで短く知らせ、クリックだけでショップRegexをクリップボードへコピーできるようにする。

## 表示条件

- PoE1であること。
- 現在エリアが既存の町エリア一覧に含まれること。
- PoBのジェム取得計画があり、現在ActのショップRegexが空でないこと。
- クエスト報酬の除外設定や短縮語上書きは、既存のRegex生成と同じ設定をそのまま使う。

条件を満たさない場合、案内は非表示にする。町から出た時も非表示にする。

## UIと操作

- みになび本文の下部に、既存ガイド本文と分けたクリック可能なラベルを置く。
- 表示文言は `💎 ショップでジェム購入可 — クリックでRegexをコピー` とする。
- 左クリック時は既存の `copy_gem_shop_search_query` と同じくQtクリップボードへ現在ActのRegexだけを保存する。ゲームへのキー入力・前面化は行わない。
- コピー後はラベルを1.2秒だけ `💎 Regexをコピーしました` に変え、元の案内へ戻す。
- みになびのロック中クリック透過設定を変更しない。クリック可能なのは既存どおり操作できる状態だけにする。

## 更新契機

- 町へ入った時、町から通常エリアへ移った時。
- PoBインポート・クリア、手動／自動Act切替、クエスト報酬除外・短縮語上書きの設定保存。
- みになびのコンテンツ更新時。

## テスト

- 町・PoE1・対象Regexありの時だけ案内テキストを返す純粋関数をテストする。
- 対象なし、町以外、PoE2では非表示になることをテストする。
- クリックでRegexをコピーし、キー入力処理を呼ばないことをQtテストする。
